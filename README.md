Bio-Trek

Bio-Trek is a Star Trek–themed biography browser. It lets you explore character and actor biographies from across the Star Trek universe through a clean, card-based web interface powered by structured JSON data and AI-generated portrait images.

Scope

Includes biographies for: TOS, TNG, DS9, VOY, ENT, DIS, PIC, SNW

Excludes (no plans to add): TAS, LD, PRO

Features

Series browsing – Switch between multiple series from a dedicated menu; each series has its own set of crew cards and portraits.

Character biographies – In-universe profiles covering origins, service history, key missions, relationships, and notable episodes.

Actor biographies – Real-world profiles covering background, Star Trek roles, and other notable work.

Card-based gallery – Each character appears as a card with portrait, character name/title, and actor name, arranged in responsive rows.

Linked views – Jump between a character’s biography and the corresponding actor’s biography from the same interface.

JSON-driven library – Biographies are stored in structured JSON files grouped by series, making the data easy to extend and reuse.

Project Status

Complete: TOS, TNG, DS9

In progress: VOY, ENT

All portraits and biographies are AI generated.
